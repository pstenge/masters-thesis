distances differ (c_ik=0). Mirrors the pairwise indicator definition.

 5. IRR Algorithm Flowchart [§3.3.5 complement]
Algorithmic flowchart complementing the existing irr-funnel figure:
Start → LHS sample N configs → Run on K datasets → all K succeed?
  No → cost=∞, discard. Yes → compute J(θ)=mean E_k(θ) → rank survivors
→ keep top P% → compute P10/P90 bounds → update Ω_{j,i+1}
→ last phase? No → next phase. Yes → output Θ*.
